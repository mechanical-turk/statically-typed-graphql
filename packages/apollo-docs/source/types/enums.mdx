Use the `t.enum` type factory to create enums:

```ts
const Membership = t.enum({
  name: 'Membership',
  values: {
    free: null,
    paid: null,
    enterprise: null,
  },
});
```

```ts
/** TypeScript */
type Membersip = 'free' | 'paid' | 'enterprise';
```

```graphql
# GraphQL
enum Membership {
  free
  paid
  enterprise
}
```
