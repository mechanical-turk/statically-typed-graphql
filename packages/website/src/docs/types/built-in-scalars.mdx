`uniform-graphql` ships with 5 built-in scalars:

|             | TypeScript         | GraphQL    |
| ----------- | ------------------ | ---------- |
| `t.string`  | `string`           | `String!`  |
| `t.float`   | `number`           | `Float!`   |
| `t.int`     | `number`           | `Int!`     |
| `t.id`      | `string \| number` | `ID!`      |
| `t.boolean` | `boolean`          | `Boolean!` |
